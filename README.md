# Hertha BSC Website ⚽

Eine kleine HTML-Website über den Berliner Fußballverein **Hertha BSC**.

## 📄 Inhalt der Website

Die Seite besteht aus mehreren HTML-Seiten:

* **Home** – Allgemeine Informationen über Hertha BSC
* **Geschichte** – Vereinsgeschichte
* **Spieler** – Bekannte Spieler des Vereins
* **Stadion** – Informationen über das Olympiastadion
* **Kontakt** – Kontaktformular
* **Impressum** – Rechtliche Angaben

## 🛠 Verwendete Technologien

* HTML5
* einfache Navigation mit Hover-Animation

## 📁 Projektstruktur

```
hertha-website/
│
├── hertha.html
└── README.md
```

## 🚀 Projekt starten

1. Projekt herunterladen oder klonen
2. `hertha.html` im Browser öffnen
