Hertha BSC Webseite

Dieses Projekt enthält die HTML-Datei `hertha.html`, die eine kleine Informationsseite über **Hertha BSC**, den Fußballverein aus Berlin, darstellt.

📂 Projektstruktur

```
hertha/
│
├── hertha.html      # Hauptseite des Projekts
├── style.css        # (optional) Stylesheet für die Seite
└── images/          # (optional) Bilder des Vereins
```

🚀 Nutzung

1. Klone das Repository:

```bash id="ly69pl"
git clone https://github.com/dein-benutzername/hertha.git
```

2. Wechsle ins Projektverzeichnis:

```bash id="wt0d9b"
cd hertha
```

3. Öffne die Seite im Browser:

```bash id="bgn2k3"
xdg-open hertha.html    # Linux
open hertha.html        # macOS
```

✏️ Anpassungen

* Passe den Inhalt von `hertha.html` nach Belieben an.
* Füge bei Bedarf ein Stylesheet `style.css` hinzu, um das Design zu verbessern.
* Bilder kannst du im Ordner `images/` ablegen und in der HTML einbinden.

⚡ Features (optional)

* Informationen über Spieler und Vereinsgeschichte
* Bildergalerie
* Kontaktformular
